    # 第一题 剑指 Offer 09. 用两个栈实现队列 (opens new window)

    用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 )

    示例 1：

    输入：
["CQueue","appendTail","deleteHead","deleteHead"]
[[],[3],[],[]]
输出：[null,null,3,-1]
    1
    2
    3
    4

    示例 2：

    输入：
["CQueue","deleteHead","appendTail","appendTail","deleteHead","deleteHead"]
[[],[],[5],[2],[],[]]
输出：[null,-1,null,null,5,2]
    1
    2
    3
    4

    提示：

    • 1 <= values <= 10000
    • 最多会对 appendTail、deleteHead 进行 10000 次调用 来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/yong-liang-ge-zhan-shi-xian-dui-lie-lcof

    解题思路：首先栈的特点为先进后出，队列的特点是先进先出。当我们向栈插入元素后，最先进入的元素处于栈的最下方。要使最下方的元素处在下一次出栈的位置，因此可以将栈一中的元素全部出栈，放入栈二中，然后将栈二出栈一个元素即可。如下图：

    代码实现：

    var CQueue = function() {
    this.stackIn = [];
    this.stackOut = [];
};

CQueue.prototype.appendTail = function(value) {
    this.stackIn.push(value);
};

CQueue.prototype.deleteHead = function() {
    // 判断out栈是否有元素，没有元素才能从in栈里面拿
    if(this.stackOut.length === 0) {
        while(this.stackIn.length !== 0) {
            this.stackOut.push(this.stackIn.pop());
        }
    }

    return this.stackOut.length === 0 ? -1 : this.stackOut.pop();
};

    # 第二题 剑指 Offer 30. 包含min函数的栈 (opens new window)

    定义栈的数据结构，请在该类型中实现一个能够得到栈的最小元素的 min 函数在该栈中，调用 min、push 及 pop 的时间复杂度都是 O(1)。

    示例:

    MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.min();   --> 返回 -3.
minStack.pop();
minStack.top();      --> 返回 0.
minStack.min();   --> 返回 -2.

    提示：

    • 各函数的调用总次数不超过 20000 次

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/bao-han-minhan-shu-de-zhan-lcof

    解题思路：在JavaScript中数组本身就是一个类似栈的数据类型，因此push、pop直接调用数组本身的方法即可。关键在于要实现时间复杂度为O(1)的min函数。我们可以在此引入一个minStack的属性来记录数组中当前的最小值。

    代码实现:

    var MinStack = function() {
    this.stack = [];
    this.minStack = [];
};

MinStack.prototype.push = function(x) {
    // 为第一个元素则必然是最小值，将小于等于最小值均入栈
    if(this.stack.length === 0 || x <= this.min()){
        this.minStack.push(x);
    }

    this.stack.push(x);
};

MinStack.prototype.pop = function() {
    // 如果删除了最小值，则应该将其出栈
    if(this.stack.pop() === this.min()) {
        this.minStack.pop();
    }
};

MinStack.prototype.top = function() {
    return this.stack[this.stack.length - 1];
};

MinStack.prototype.min = function() {
    return this.minStack[this.minStack.length - 1];
};
