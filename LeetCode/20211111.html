    # 第一题 剑指 Offer 46. 把数字翻译成字符串 (opens new window)

    给定一个数字，我们按照如下规则把它翻译为字符串：0 翻译成 “a” ，1 翻译成 “b”，……，11 翻译成 “l”，……，25 翻译成 “z”。一个数字可能有多个翻译。请编程实现一个函数，用来计算一个数字有多少种不同的翻译方法。

    示例 1:

    输入: 12258
输出: 5
解释: 12258有5种不同的翻译，分别是"bccfi", "bwfi", "bczi", "mcfi"和"mzi"
    1
    2
    3

    提示：

    • 0 <= num < 2^31

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/ba-shu-zi-fan-yi-cheng-zi-fu-chuan-lcof

    解题思路：按照题解思路，列出递推方程，f(n)代表长度为n的数字能够翻译成的字符串个数。x为xn-1和xn组成的字符串。

    f(n) = f(n-1) + f(n-2)(x能组成双字符)

    f(n) = f(n-1)(x不能组成双字符)

    并且递推初始条件为f(0) = f(1) = 1。

    注意：

    为什么f(0) = 1，因为f(2) = f(1) + f(0) = 2（假设满足条件：x能够组成双字符），所以f(0) = 1。

    并且既可以从右边开始遍历，也可以从左边开始遍历，二者结果一样。

    还可以使用滚动数组的方法，仅需要记录f(n)、f(n-1)、f(n-2)

    代码实现：

    var translateNum = function(num) {
    let p=1, q=1, r=1;

    while(num) {
        if(num % 100 > 9 && num % 100 < 26) {
            r = p + q;
        } else {
            r = q;
        }

        p = q;
        q = r;
        num = Math.floor(num / 10);
    }

    return r;
};

    # 第二题 剑指 Offer 48. 最长不含重复字符的子字符串 (opens new window)

    请从字符串中找出一个最长的不包含重复字符的子字符串，计算该最长子字符串的长度。

    示例 1:

    输入: "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

    2
    3

    示例 2:

    输入: "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
    1
    2
    3

    示例 3:

    输入: "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

    提示：

    • s.length <= 40000

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/zui-chang-bu-han-zhong-fu-zi-fu-de-zi-zi-fu-chuan-lcof

    解题思路：动态规划 / 双指针 + 哈希表，清晰图解 (opens new window)

    **注意：**为什么说当dp[j - 1] < j - i，说明字符s[i]在子字符串dp[j-1]区间之外？j - i代表s[j]左边距离最近的相同字符到当前字符的范围距离，而dp[j - 1]代表了以上一个字符结尾的最长不含重复字符的子字符串长度，如果小于j - i，说明以上一个字符结尾的最长不含重复字符的子字符串长度小于s[j]左边距离最近的相同字符到当前字符的范围距离，所以不包含s[i]。

    代码实现：

    var lengthOfLongestSubstring = function(s) {
    let hashmap = new Map();
    let max=0, tem=0;

    for(let i=0; i<s.length; i++) {
        let idx = hashmap.get(s[i]);
        if(idx === undefined) {
            idx = -1;
        }
        hashmap.set(s[i], i);

        if (tem < i - idx) {
            tem++;
        } else {
            tem = i - idx;
        }

        if(tem > max) {
            max = tem;
        }
    }

    return max;
};
