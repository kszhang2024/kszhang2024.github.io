    # 第一题 剑指 Offer 26. 树的子结构 (opens new window)

    输入两棵二叉树A和B，判断B是不是A的子结构。(约定空树不是任意一个树的子结构)

    B是A的子结构， 即 A中有出现和B相同的结构和节点值。

    例如: 给定的树 A:

         3
    / \
   4   5
  / \
 1   2

    给定的树 B：

       4 
  /
 1
    1
    2
    3

    返回 true，因为 B 与 A 的一个子树拥有相同的结构和节点值。

    示例 1：

    输入：A = [1,2,3], B = [3,1]
输出：false
    1
    2

    示例 2：

    输入：A = [3,4,5,1,2], B = [4,1]
输出：true
    1
    2

    限制：

    0 <= 节点个数 <= 10000
    1

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/shu-de-zi-jie-gou-lcof

    解题思路：这题可以遍历A树的节点，依次判断当前节点的子树是否包含了B树的结构。

    B树当前节点的值和A子树当前的节点的值不同，那么存在差异节点，结构不同，返回false。

    A子树的值为null，说明A子树已经遍历完，但是B树未遍历完，结构不同，返回false。

    B树的值为null，说明B树的值已经遍历完，未出现与A子树有差异的节点，B树的结构包含在了当前A树节点的子树中。

    注意：recur最后必须是并运算，A、B树的左子树和A、B树的右子树均满足A包含了B的结构，才能说当前A树包含了B树的结构。

    代码实现：

    var isSubStructure = function(A, B) {
    // A或者B为空树时，B均不可能为A的子结构
    if (A === null || B === null) {
        return false;
    }
	
    // 判断当前A、B树的根节点是否满足子结构，否则判断A的左子树结构是否和B满足子结构，否则判断A的右子树结构
    return recur(A, B) || isSubStructure(A.left, B) || isSubStructure(A.right, B);
};

function recur(A, B) {
    // 当B为空树时，说明B树当前值不存在，满足A的子结构
    if(B === null) {
        return true;
    }
    
	// 当A为空树时，说明A树已经遍历完，而B树还未完。
    // 当A节点当前值和B节点当前值不等
    // 两种情况均不满足B是A的子结构
    if(A === null || A.val !== B.val) {
        return false;
    }
	
    // 继续判断A、B树的左子树和右子树
    return recur(A.left, B.left) && recur(A.right, B.right); 
}

    # 第二题 剑指 Offer 27. 二叉树的镜像 (opens new window)

    请完成一个函数，输入一个二叉树，该函数输出它的镜像。

    例如输入：

     	 4
   /   \
  2     7
 / \   / \
1   3 6   9

    镜像输出：

     	 4
   /   \
  7     2
 / \   / \
9   6 3   1

    示例 1：

    输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]
    1
    2

    限制：

    0 <= 节点个数 <= 1000
    1

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/er-cha-shu-de-jing-xiang-lcof

    解题思路：镜像输出就是将该二叉树的左右子树互换。看到此题，我们首先回忆一下二叉树先序遍历的顺序为根节点、左子树、右子树。我们尝试一下遍历顺序为根节点、右子树、左子树，就相当于把左子树当成了右子树。后来发现这么做是有问题的，因为不同的二叉树，先序遍历的顺序是有可能相同的，所以返回的镜像即使遍历顺序一致，但它有可能是与此树相同先序遍历顺序的二叉树的镜像。而且此题必须要返回树的根节点，所以前一种方法不能用于此题。那么就老老实实按照定义，将左子树和右子树互换，返回根节点吧。

    **注意：**为什么要处理根节点为null的情况呢？因为根节点为null时不能读取左子树和右子树，会报错。

    代码实现：

    var mirrorTree = function(root) {
    if(root === null) {
        return root;
    }
    
    // 交换左右子树
    let temp = root.left;
    root.left = root.right;
    root.right = temp;
    
    // 递归处理左子树和右子树
    mirrorTree(root.left);
    mirrorTree(root.right);
    return root;
};

    # 第三题 剑指 Offer 28. 对称的二叉树 (opens new window)

    请实现一个函数，用来判断一棵二叉树是不是对称的。如果一棵二叉树和它的镜像一样，那么它是对称的。

    例如，二叉树 [1,2,2,3,4,4,3] 是对称的。

        1
   / \
  2   2
 / \ / \
3  4 4  3

    但是下面这个 [1,2,2,null,3,null,3] 则不是镜像对称的:

        1
   / \
  2   2
   \   \
   3    3

    示例 1：

    输入：root = [1,2,2,3,4,4,3]
输出：true
    1
    2

    示例 2：

    输入：root = [1,2,2,null,3,null,3]
输出：false
    1
    2

    限制：

    0 <= 节点个数 <= 1000
    1

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/dui-cheng-de-er-cha-shu-lcof

    解题思路：此题和第二题均为镜像问题，可以把该问题看作镜像问题的扩展。所以暴力将镜像求出来再比较就是最先想出来的方法。镜像就是将左子树和右子树互换，那么我们应该去比较左子树和右子树的值，左子树的左子树和右子树的右子树的值，左子树的右子树和右子树的左子树的值。那么我们仅需要将对应节点传入进去，再进行比较。

    如果二者值不同或者有一方为空则返回false，如果两者同时为空，那么说明递归结束，到目前位置均满足镜像对称，返回true。二者均不为空，且值相等，那么说明该比较左子树的左子树和右子树的右子树、左子树的右子树和右子树的左子树。并且两边值均要相等才能够返回true，用&&连接。

    代码实现：

    var isSymmetric = function(root) {
    if(root === null) {
        return true;
    }

    return recur(root.left, root.right);
};

function recur(L, R) {
    if(L === null && R === null) {
        return true;
    }
    
    // 注意null判断所放位置，否则读取val属性会报错。
    if(L === null || R === null || L.val !== R.val) {
        return false;
    }

    return recur(L.left, R.right) && recur(L.right, R.left);
}

    # 总结

    今天的题也均和二叉树有关，大多都是镜像、对称问题。主要考察了递归的知识点，递归需要注意的就是递归终止条件，以及下一步要干什么，刚开始刷题有点儿不适应递归，第一题和第三题均参考了题解的解法，以后还需要多刷题，熟悉递归的思路。如果有什么不对的地方，或者有更好的方案，欢迎提出，大家一起讨论。
