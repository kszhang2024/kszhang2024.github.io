    # 第一题 剑指 Offer 34. 二叉树中和为某一值的路径 (opens new window)

    给你二叉树的根节点 root 和一个整数目标和 targetSum ，找出所有从根节点到叶子节点路径总和等于给定目标和的路径。

    叶子节点 是指没有子节点的节点。

    示例 1：

    输入：root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22
输出：[[5,4,11,2],[5,8,4,5]]
    1
    2

    示例 2：

    输入：root = [1,2,3], targetSum = 5
输出：[]
    1
    2

    示例 3：

    输入：root = [1,2], targetSum = 0
输出：[]
    1
    2

    提示：

    • 树中节点总数在范围 [0, 5000] 内
    • -1000 <= Node.val <= 1000
    • -1000 <= targetSum <= 1000

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/er-cha-shu-zhong-he-wei-mou-yi-zhi-de-lu-jing-lcof

    解题思路：经过昨天题目的洗礼，终于开始有一点点悟了。对于这种搜索与回溯算法的题目，一般的解题思路就是利用DFS、BFS。所以对于这题，一下就想到了DFS。将当前路径，路径和存储下来，然后到了叶子节点判断是否和target相等，相等就将该条路径存储下来。成功解决题目，但是效率不高，所以开始阅读题解。题解和我的思路大致一致，但在判断是否等于target的时候，是采用每一次减去路径节点值，减到最后叶子节点的时候，target是否等于0来判断的。

    **注意：**记录路径的时候，一定要记录该条路径的拷贝，否则在下个节点记录路径的时候会影响当前路径的情况。

    代码实现：

    // DFS
var pathSum = function(root, target) {
    function dfs(root, sum, path) {
        if(root.left === null && root.right === null) {
            if(sum + root.val === target) {
                // 路径和 与 目标值相同。
                path.push(root.val);
                res.push(path);
            }

            return;
        }

        sum += root.val;
        // 记录路径
        path.push(root.val);
        if(root.left !== null) dfs(root.left, sum, [...path]);
        if(root.right !== null) dfs(root.right, sum, [...path]);
    }
    
	// 空节点处理
    if(root === null) {
        return [];
    }
    
    let res = [];
    dfs(root, 0, []);
    return res;
};

// 题解 DFS
var pathSum = function(root, target) {
    function dfs(root, sum) {
        if(root === null) {
            return;
        }

        path.push(root.val);
        sum -= root.val;
        if(root.left === null && root.right === null && sum === 0) {
            	// 防止后续对 path 的更改造成之前路径的改变。
                res.push([...path]);
        }

        dfs(root.left, sum);
        dfs(root.right, sum);
        // 为什么删除该节点？主要是进行回溯，当前路径遍历完毕，回溯到上一个状态，回溯到上一个节点。
        // 这样我们每次在当前节点的时候，path 记录的都是到当前节点的路径。
        path.pop();
    }

    let res=[], path=[];
    dfs(root, target);
    return res;
};

    # 第二题 剑指 Offer 36. 二叉搜索树与双向链表 (opens new window)

    输入一棵二叉搜索树，将该二叉搜索树转换成一个排序的循环双向链表。要求不能创建任何新的节点，只能调整树中节点指针的指向。

    为了让您更好地理解问题，以下面的二叉搜索树为例：

    我们希望将这个二叉搜索树转化为双向循环链表。链表中的每个节点都有一个前驱和后继指针。对于双向循环链表，第一个节点的前驱是最后一个节点，最后一个节点的后继是第一个节点。

    下图展示了上面的二叉搜索树转化成的链表。“head” 表示指向链表中有最小元素的节点。

    特别地，我们希望可以就地完成转换操作。当转化完成以后，树中节点的左指针需要指向前驱，树中节点的右指针需要指向后继。还需要返回链表中的第一个节点的指针。

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/er-cha-sou-suo-shu-yu-shuang-xiang-lian-biao-lcof

    解题思路：题目希望就地完成转换操作，所以我们直接将左、右指针当成双向链表的前、后指针。然后需要返回排序的双向链表，我们可以通过中序遍历来利用二叉排序树的特点。最后处理循环链表的特点，尾指针的右指针指向头指针，头指针的左指针指向尾指针。

    **注意：**为什么cur === null处理了空树的情况，还要在dfs外处理空树的情况？因为空树时不能读取head的左子树，会报错。

    代码实现：

    var treeToDoublyList = function(root) {
    let pre = null, head = null;

    function dfs(cur) {
        if(cur === null) {
            return;
        }

        dfs(cur.left);
        if(pre === null){
            head = cur;
        } else {
            pre.right = cur;
        }
        cur.left = pre;
        pre = cur;
        dfs(cur.right);
    }

    if(root === null) {
        return null;
    }
    dfs(root);
    head.left = pre;
    pre.right = head;
    return head;
};

    # 第三题 剑指 Offer 54. 二叉搜索树的第k大节点 (opens new window)

    给定一棵二叉搜索树，请找出其中第k大的节点。

    示例 1:

    输入: root = [3,1,4,null,2], k = 1
   3
  / \
 1   4
  \
   2
输出: 4
    1
    2
    3
    4
    5
    6
    7

    示例 2:

    输入: root = [5,3,6,2,4,null,null,1], k = 3
       5
      / \
     3   6
    / \
   2   4
  /
 1
输出: 4
    1
    2
    3
    4
    5
    6
    7
    8
    9

    限制：

    • 1 ≤ k ≤ 二叉搜索树元素个数

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/er-cha-sou-suo-shu-de-di-kda-jie-dian-lcof

    解题思路：这题和上一题差不多，这题要求返回第k大的元素，而二叉搜索树的特点如下

    • 若任意节点的左子树不空，则左子树上所有节点的值均小于它的根节点的值；
    • 若任意节点的右子树不空，则右子树上所有节点的值均大于它的根节点的值；
    • 任意节点的左、右子树也分别为二叉查找树；
    • 没有键值相等的节点。

    可以利用中序遍历，但是这里中序遍历需要做点改变，我们先遍历右子树，然后遍历根节点，再遍历左子树，这样就是从大到小排序的顺序遍历，最后记录第k个元素就是第k大的元素。

    注意：

    代码实现：

    // DFS
var kthLargest = function(root, k) {
    let count = 0;
    let res = null;

    function dfs(cur) {
        if(cur === null) {
            return;
        }

        dfs(cur.right);
        count++;
        if(count === k) {
            res = cur.val;
        }
        dfs(cur.left);
    }

    dfs(root);
    return res;
};
