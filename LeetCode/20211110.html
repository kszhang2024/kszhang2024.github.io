    # 第一题 剑指 Offer 42. 连续子数组的最大和 (opens new window)

    输入一个整型数组，数组中的一个或连续多个整数组成一个子数组。求所有子数组的和的最大值。

    要求时间复杂度为O(n)。

    示例1:

    输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
    1
    2
    3

    提示：

    • 1 <= arr.length <= 10^5
    • -100 <= arr[i] <= 100

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/lian-xu-zi-shu-zu-de-zui-da-he-lcof

    解题思路：这是首先可以思考一下暴力法，那么暴力法是如何做的呢？求出每一个子数组的和，记录最大值，然后返回最大值。利用双重循环就解决了。题目要求时间复杂度为O(n)，并且这道题也是经典的动态规划问题，所以在此利用动态规划来求解。假设F(n)表示以第 i 个数结尾的「连续子数组的最大和」。

    F(n) = max(F(n-1) + n, n)

    F(arr[0]) = arr[0]

    **注意：**为什么没有使用dp数组保留第 i 个数结尾的「连续子数组的最大和」呢？这是因为我们最后只需要这个dp数组中的最大值，就可以使用maxVal这个变量把最大值存储下来即可，使空间复杂度为O(1)。

    代码实现：

    var maxSubArray = function(nums) {
    let pre = 0, maxVal = -1000000;
    
    for (let num of nums) {
        pre = Math.max(pre + num, num);
        // 保留连续子数组的最大和
        maxVal = Math.max(pre, maxVal);
    }
    
    return maxVal;
};

    # 第二题 剑指 Offer 47. 礼物的最大价值 (opens new window)

    在一个 mn 的棋盘的每一格都放有一个礼物，每个礼物都有一定的价值（价值大于 0）。你可以从棋盘的左上角开始拿格子里的礼物，并每次向右或者向下移动一格、直到到达棋盘的右下角。给定一个棋盘及其上面的礼物的价值，请计算你最多能拿到多少价值的礼物？*

    示例 1:

    输入: 
[
  [1,3,1],
  [1,5,1],
  [4,2,1]
]
输出: 12
解释: 路径 1→3→5→2→1 可以拿到最多价值的礼物

    提示：

    • 0 < grid.length <= 200
    • 0 < grid[0].length <= 200

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/li-wu-de-zui-da-jie-zhi-lcof

    解题思路：我们设dp(i, j)表示(i, j)位置的最大值，然后可列出递推方程，当前位置最大值等于左方值和上方值中的的最大值加上当前值。

    dp(i, j) = max(dp(i-1, j), dp(i, j-1)) + mat[i, j]

    dp(i, 0) = 0

    dp(0, j) = 0

    **注意：**在原数组上修改可以减小空间复杂度，使其为O(1)。

    代码实现：

    var maxValue = function(grid) {
    for(let i=0; i<grid.length; i++) {
        for(let j=0; j<grid[i].length; j++) {
            if((i > 0 && j > 0)) {
                grid[i][j] += Math.max(grid[i-1][j], grid[i][j-1]);
            } else if ((i > 0) && (j === 0)) {
                grid[i][j] += grid[i-1][j];
            } else if ((i === 0) && (j > 0)) {
                grid[i][j] += grid[i][j-1];
            }
        }
    }

    return grid[grid.length - 1][grid[0].length - 1];
};
